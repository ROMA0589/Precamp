# BackpackButton

Plugin para Rust que agrega un botón en la interfaz para abrir la mochila usando Backpacks v2.x.

## Características
- Botón cuadrado con imagen de mochila, visible en pantalla.
- Compatible con Rust 2275 y Backpacks v2.1.8.
- Permite abrir la mochila con un solo clic.

## Instalación
1. Copia `BackpackButton.cs` en la carpeta `oxide/plugins` de tu servidor Rust.
2. Recarga el plugin con el comando:
   ```
   oxide.reload BackpackButton
   ```

## Configuración
Puedes ajustar la posición, tamaño y color del botón en el archivo de configuración `BackpackButton.json`.

## Requisitos
- Rust 2275
- Backpacks v2.1.8
