
# Precamp

Precamp is an educational web platform designed to support spiritual and personal development for young people. The site offers interactive lessons, games, activities, and resources based on Spiritism and values education.

## Features
- Interactive lessons and activities
- Educational games (memory, quiz, etc.)
- Video and multimedia resources
- Responsive and modern design
- Easy navigation and user experience

## Technologies Used
- Next.js (React framework)
- TypeScript
- CSS Grid and Flexbox
- Vercel (deployment)

## Installation
1. Clone this repository:
	```
	git clone https://github.com/ROMA0589/Precamp.git
	```
2. Install dependencies:
	```
	npm install
	```
3. Run the development server:
	```
	npm run dev
	```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage
- Visit [precamp.com.br](https://precamp.com.br) for the live site.
- Explore lessons, games, and activities from the main menu.
- For contributors: edit content in the `src/pages` and `src/components` folders.
